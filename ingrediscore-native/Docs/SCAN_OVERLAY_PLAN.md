# Scan Overlay Plan

## Barcode scan overlay
Should include:
- central scan frame
- subtle instruction text
- cancel/back affordance
- loading indicator after successful read

## Ingredient scan overlay
Should include:
- rectangular guidance frame optimized for ingredient labels
- short instruction text
- capture button
- retake flow after capture
- optional tip text for glare / blur

## Design goals
- minimal clutter
- high contrast
- clear success state
- fast retake path
